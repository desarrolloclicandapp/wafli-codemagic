Política de Privacidad

Última actualización: mayo 2026

Esta Política de Privacidad explica qué datos trata WaFli, para qué los usa, con quién puede compartirlos y qué controles tienes sobre tu información. Para consultas de privacidad puedes escribir a soporte@wafli.ai.

WaFli es un wingman de conversación. No es una red social abierta, no es una app de citas y no envía mensajes automáticamente por el usuario.

1. Datos que podemos tratar

Cuenta e identidad

Email, identificadores internos, login social, tokens de sesión, estado de cuenta, preferencias, idioma, tono, país o variante de español, onboarding y aceptación legal.

Conversaciones y contexto

Chats, mensajes recientes, metadatos de conversación, nombres visibles, estado de lectura, archivos o medios recientes y datos técnicos necesarios para mostrar la app y generar sugerencias cuando lo solicitas.

Uso, cuota y pagos

Eventos de uso de IA, consumo de cuota, plan, packs, transacciones, estado de suscripción y datos de facturación procesados por proveedores externos.

Notificaciones y dispositivo

Tokens push, suscripciones web push, preferencias de notificación, identificadores de app, plataforma, versión, eventos de pantalla, diagnóstico técnico y datos de soporte.

2. IA y minimización

WaFli envía contexto limitado a proveedores de IA únicamente cuando pides una acción como sugerir, reescribir, analizar, abrir o reactivar. No activamos el modelo en segundo plano para leer tus chats sin una acción tuya. Las respuestas de IA son borradores editables y pueden ser incorrectas o inadecuadas.

Los eventos de Analytics no deben incluir contenido privado de mensajes, números completos, archivos ni contenido sensible de conversaciones.

3. Finalidades

Usamos datos para autenticarte, prestar la app, conectar conversaciones, generar borradores de IA, gestionar cuota y pagos, enviar notificaciones, mejorar estabilidad, medir uso agregado, prevenir abuso, proteger seguridad, cumplir obligaciones legales y ofrecer soporte.

4. Proveedores

Podemos usar proveedores como OpenAI para funciones de IA, Google/Firebase para notificaciones y Analytics, Google o Apple para login, Stripe o los sistemas de pago de las tiendas para pagos, servicios de hosting, email y herramientas de soporte. Estos proveedores actúan para prestar funciones de la app, procesar pagos, seguridad, analítica o soporte.

No vendemos tus datos personales.

5. Analytics

Usamos Firebase/Google Analytics para entender pantallas, eventos de producto, activación de funciones, estabilidad y uso general. Analytics se configura para medir producto, no para registrar contenido privado de chats. Actualmente no usamos datos de WaFli para publicidad personalizada ni para rastreo publicitario entre apps o sitios de terceros.

6. Notificaciones

Si activas notificaciones, registramos tokens push o suscripciones para avisos de mensajes, cuota, estado de conexión, pagos o novedades. Puedes desactivarlas desde Ajustes o desde el sistema operativo. Si eliminas tu cuenta, eliminamos o desactivamos tokens push asociados a tu usuario.

7. Pagos

WaFli no almacena directamente datos completos de tarjeta. Los pagos se procesan mediante proveedores externos como Stripe o los sistemas de pago requeridos por App Store y Google Play cuando corresponda.

8. Seguridad

Usamos medidas razonables como HTTPS, control de acceso, minimización, separación de secretos, revocación de sesiones y controles técnicos. Ningún sistema es perfecto; si detectamos una incidencia relevante, actuaremos conforme a la ley aplicable.

9. Retención y eliminación

Conservamos datos mientras tu cuenta esté activa o mientras sea necesario para prestar el servicio. El cache de conversaciones debe ser limitado y puede eliminarse desde Ajustes. Al solicitar eliminación de cuenta, desconectamos la mensajería conectada, revocamos sesiones, eliminamos cache de conversaciones y eliminamos o desactivamos tokens push. Podemos conservar registros mínimos durante un periodo limitado por seguridad, prevención de fraude, cumplimiento legal, disputas o respaldo.

Puedes consultar el recurso público de eliminación en /account-deletion.html.

10. Tus controles

Puedes solicitar acceso, exportación, corrección, eliminación, desconexión de la mensajería conectada y borrado de historial desde Ajustes o escribiendo a soporte@wafli.ai. Algunas leyes pueden darte derechos adicionales según tu país.

11. Edad e información sensible

WaFli está pensada para personas adultas. No debes usar la app para tratar información de terceros sin una base legítima. Tampoco debes usarla para emergencias, asesoramiento médico, legal, financiero, psicológico o de seguridad personal.

12. Transferencias internacionales

WaFli y sus proveedores pueden procesar datos en Estados Unidos y otros países. Cuando sea necesario, usaremos mecanismos legales adecuados para transferencias internacionales.

13. Cambios

Podemos actualizar esta política para reflejar cambios de producto, legales o técnicos. Publicaremos la fecha de última actualización y, si el cambio es material, podremos pedir aceptación o mostrar aviso dentro de la app.

14. Contacto

Para soporte, privacidad o consultas legales: soporte@wafli.ai.
