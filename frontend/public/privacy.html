Política de Privacidad

Última actualización: 15 de junio de 2026

Esta política explica cómo WaFli trata los datos necesarios para ofrecer su app de asistencia conversacional, incluyendo cuenta, preferencias, conversaciones conectadas por el usuario, herramientas manuales, funciones de IA, notificaciones, pagos, soporte, analítica y seguridad.

1. Contacto de privacidad

Para ejercer derechos de privacidad, solicitar soporte o pedir eliminación de cuenta o datos, escribe a soporte@wafli.ai.

2. Datos que podemos tratar

  • Datos de cuenta: email, nombre o alias si el proveedor de acceso lo entrega, identificadores internos, proveedor de inicio de sesión, estado de onboarding y aceptaciones legales.
  • Preferencias: idioma, variante de español, ajustes de estilo, notificaciones, privacidad y estado de conexión.
  • Conversaciones conectadas: chats, contactos, nombres visibles, mensajes recientes, metadatos y archivos o medios necesarios para mostrar la app y preparar sugerencias cuando el usuario solicita una acción.
  • Herramientas manuales: texto que el usuario pega, mensaje a responder, contexto adicional, capturas adjuntas y borradores generados.
  • Datos de IA: instrucciones, objetivo elegido, contexto relevante, resultado generado, consumo de cuota y registros técnicos mínimos para prestar la función.
  • Pagos y suscripciones: plan, producto comprado, estado de suscripción, fechas, identificadores de transacción y validación de compras. WaFli no almacena datos completos de tarjeta.
  • Notificaciones y dispositivo: tokens push, plataforma, preferencias y datos técnicos mínimos para enviar avisos solicitados.
  • Analítica, rendimiento y seguridad: interacciones con la app, diagnósticos, errores, dirección IP, agente de usuario y señales antifraude.
  • Soporte: mensajes, capturas o información que el usuario decida compartir para resolver incidencias.

3. Uso de IA, OpenAI y consentimiento

WaFli usa OpenAI como proveedor externo de inteligencia artificial para generar respuestas, reescribir textos, analizar mensajes o ayudar con herramientas manuales. WaFli no comparte chats con OpenAI en segundo plano ni monitoriza conversaciones continuamente.

Antes de enviar datos a OpenAI, la app muestra un aviso dentro de WaFli que explica qué datos pueden enviarse, identifica a OpenAI como proveedor externo de IA y pide permiso explícito para esa generación. Si el usuario cancela, WaFli no envía nada a OpenAI y la generación no continúa.

Cuando el usuario autoriza una acción de IA, WaFli puede enviar a OpenAI únicamente los datos necesarios para esa acción: mensaje seleccionado o pegado, contexto reciente relevante, instrucciones u objetivo, contexto adicional escrito por el usuario y capturas o imágenes adjuntadas voluntariamente. OpenAI procesa esos datos para devolver el resultado solicitado.

Las respuestas generadas son borradores. WaFli no envía mensajes automáticamente; el usuario debe revisar, editar, copiar o descartar cada resultado.

4. Para qué usamos los datos

Usamos los datos para autenticar usuarios, prestar la app, mostrar conversaciones conectadas, generar borradores editables con IA cuando el usuario lo solicita, gestionar cuotas y pagos, enviar notificaciones, ofrecer soporte, mejorar estabilidad, prevenir abuso, proteger la seguridad, personalizar preferencias y cumplir obligaciones legales.

5. Proveedores y terceros

Podemos compartir datos estrictamente necesarios con proveedores de infraestructura, autenticación, IA, analítica, notificaciones, soporte, procesamiento de pagos, validación de suscripciones y cumplimiento legal. Entre ellos pueden estar OpenAI para funciones de IA, Google/Firebase para analítica, notificaciones e infraestructura, Apple o Google para inicio de sesión y compras dentro de la app, RevenueCat para validación de suscripciones, Stripe para pagos web cuando aplique y proveedores de hosting o monitoreo.

Exigimos que los proveedores que tratan datos personales ofrezcan protecciones iguales o equivalentes a las descritas en esta política y usen los datos solo para prestar los servicios contratados.

6. Seguridad

Aplicamos cifrado en tránsito mediante HTTPS/TLS, controles de acceso, separación de credenciales, registros de seguridad y minimización de datos. Trabajamos para limitar el acceso y reducir los datos persistidos a lo necesario para funcionalidad, soporte, seguridad y cumplimiento.

7. Retención y eliminación

Conservamos datos mientras la cuenta esté activa o mientras sean necesarios para prestar el servicio. El usuario puede borrar historial en caché, desconectar conversaciones conectadas y solicitar eliminación de cuenta o datos desde la app o escribiendo a soporte@wafli.ai. La página pública Eliminación de cuenta y datos detalla plazos y excepciones legales.

8. Controles del usuario

Según la ubicación del usuario, puede solicitar acceso, exportación, corrección, eliminación, oposición o limitación del tratamiento. También puede rechazar el envío de datos a OpenAI cancelando el aviso de permiso antes de cada acción de IA.

9. Menores

WaFli está dirigido a personas adultas. No está diseñado para menores de 18 años ni para usuarios por debajo de la mayoría de edad aplicable en su jurisdicción.

10. Transferencias internacionales

WaFli y sus proveedores pueden procesar datos en países distintos al del usuario, incluyendo Estados Unidos y la Unión Europea. Cuando corresponda, usamos mecanismos legales adecuados y medidas contractuales o técnicas para proteger la información.

11. Cambios

Podemos actualizar esta política cuando cambien la app, los proveedores, las obligaciones legales o las prácticas de tratamiento. La versión vigente se publicará en esta página con su fecha de actualización.
